# Design Guidelines

**Overall Theme:** Dark, futuristic, minimal.

**Colors (initial draft):**
- Background: #050816
- Card background: #0f172a
- Accent (primary): #22c55e
- Accent (secondary): #38bdf8
- Text: #e5e7eb

**Font Choices:**
- Primary: Inter (or system font stack)
- Code / tech labels: JetBrains Mono or monospace

**Theme Switching:**
- Default: Dark mode
- Future: Support `prefers-color-scheme` and a toggle in the navbar.
